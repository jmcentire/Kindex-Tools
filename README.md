# Kindex-Tools

The [kindex.tools](https://kindex.tools) website — marketing and documentation for the [Kindex](https://github.com/jmcentire/kindex) knowledge graph platform.

## About Kindex

Kindex is the memory layer AI coding agents do not have. A persistent knowledge graph for Claude Code, Codex, Gemini CLI, Google Antigravity, OpenCode, Cursor, and MCP-capable workflows with 52 MCP tools, 80 CLI commands, session tags, scoped attention reminders, headless Codex/OpenCode wakeup reminders, efficient dream consolidation, five context tiers, and git-tracked `.kin` project context with portable code maps and opt-in work policy enforcement.

- **GitHub**: [jmcentire/kindex](https://github.com/jmcentire/kindex)
- **PyPI**: [kindex](https://pypi.org/project/kindex/)
- **Docs**: [kindex.tools](https://kindex.tools/) and [jmcentire.github.io/kindex](https://jmcentire.github.io/kindex/)
- **MCP Market**: [mcpmarket.com/server/kindex](https://mcpmarket.com/server/kindex)
